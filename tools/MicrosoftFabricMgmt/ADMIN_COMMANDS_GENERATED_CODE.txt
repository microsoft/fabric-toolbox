KEY PATTERNS IMPLEMENTED
========================================

All 50 commands follow these patterns:

1. BASE URL:
   $powerBIAdminBaseUrl = "https://api.powerbi.com/v1.0/myorg"

2. AUTHENTICATION:
   Invoke-FabricAuthCheck -ThrowOnFailure
   $script:FabricAuthContext.FabricHeaders

3. GET OPERATIONS:
   - Optional ID parameter for single item retrieval
   - Filter, Top, Skip for list operations
   - -Raw switch for raw API response
   - Type decoration: Select-FabricResource -TypeName 'MicrosoftFabric.AdminXxx'

4. MUTATIONS:
   [CmdletBinding(SupportsShouldProcess, ConfirmImpact = 'High')]
   if ($PSCmdlet.ShouldProcess(...)) { API call }

5. ERROR HANDLING:
   try/catch with Write-FabricLog -Level Error
